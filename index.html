Inspector - Documentación de Uso

Descripción

Este repositorio contiene los archivos base para entrenar y probar un flujo de procesamiento de documentos. La idea principal es tomar documentos de ejemplo, entrenar un modelo que los comprenda y posteriormente ejecutar consultas sobre el modelo entrenado.

Tecnologías y librerías

  • Python 3
  • Bibliotecas de procesamiento de lenguaje natural (por ejemplo, langchain)
  • Herramientas de vectorización (por ejemplo, sentence-transformers)

Cómo usar este repositorio

  1. Coloque sus documentos de ejemplo en la carpeta docs/.
  2. Ejecute python3 train.py para procesar los documentos y entrenar el modelo.
  3. Después del entrenamiento, pruebe el flujo ejecutando python3 run.py.

Agregar nuevos documentos

  1. Cree un archivo (por ejemplo, docs/ejemplo.txt) con el contenido que desea agregar.
  2. Repita el paso de entrenamiento para actualizar el modelo con la nueva información.
  3. Realice consultas de prueba utilizando el script de ejecución.

Comandos útiles

        python3 train.py   # Entrena el modelo con los documentos en docs/
        python3 run.py     # Abre un prompt para hacer preguntas al modelo
